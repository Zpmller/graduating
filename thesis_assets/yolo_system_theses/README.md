# YOLO检测加系统方向硕士论文资料

筛选口径：优先选择“YOLO/目标检测作为系统能力的一部分，论文重点在系统设计、实现、部署、告警或交互流程”，排除以“改进YOLO、注意力机制、轻量化网络、硬件加速器”为核心创新点的论文。

## 推荐使用

1. `selected/01_基于目标检测算法的智慧车库系统_吉林大学_高放_2020.pdf`
   - 题名：基于目标检测算法的智慧车库系统
   - 作者/学校：高放，吉林大学，硕士学位论文，2020
   - 适配点：目标检测算法 + 软硬件系统 + 浏览器/服务器用户界面 + 实际运行测试，结构很适合参考“系统总体设计、功能模块、实现与测试”写法。
   - 来源：https://www.printfinn.com/assets/pdf/master_degree_thesis_gao_fang.pdf

2. `selected/07_基于深度学习的地铁非法过闸机行为检测研究与实现_硕士论文.pdf`
   - 题名：基于深度学习的地铁非法过闸机行为检测研究与实现
   - 作者：万思远，硕士学位论文，2022
   - 适配点：使用YOLOv5作为行人检测链路，围绕视频事件检测、告警、结果保存等系统流程展开；不属于“改进YOLO模型”论文，适合参考安全监控类检测系统的实验与告警逻辑。
   - 来源：https://airl.csu.edu.cn/PDFs/wsy.pdf

## 候选但不推荐

这些PDF已放在 `候选_不推荐_偏模型改进/`，主要原因是论文核心贡献偏模型结构改进、轻量化、注意力机制或硬件加速，不符合“不涉及改进模型”的要求：

- `02_石河子大学_片式滴头质量检测系统_YOLO_2025.pdf`：YOLOv8轻量化与精度优化是主要贡献。
- `03_南京大学_路面垃圾检测系统_YOLO_2024.pdf`：主线是卷积神经网络视觉注意力机制。
- `04_科技小院_害虫图像识别系统_YOLOv5.pdf`：改进YOLOv5s，引入GhostNet和注意力机制。
- `05_科技小院_农业目标检测系统_YOLOv5.pdf`：加入CBAM注意力机制和小目标分支。
- `06_石河子大学_目标检测跟踪系统设计与实现_YOLO.pdf`：提出ELFN-YOLO和跟踪改进方法。
- `08_南京大学_检测系统设计实现_YOLO_2025.pdf`：核心是边缘平台高能效神经网络与加速器。
- `09_基于深度学习的移动式棉花虫害检测系统研究_硕士论文.pdf`：提出GBW-YOLOv5并做剪枝。
- `10_科技小院_实时检测系统手机端软件实现_YOLOv5.pdf`：提出改进YOLOv4与分割算法。
