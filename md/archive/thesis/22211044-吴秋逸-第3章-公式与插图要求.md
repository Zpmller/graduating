# 第 3 章 公式与插图要求

本文档列出《关键功能模块设计与实现》一章中所有公式编号、图号及绘制/排版要求，便于统一编号与制图。

---

## 一、公式清单

| 编号 | 所在小节 | 公式内容 | 说明 |
|------|----------|----------|------|
| (3-1) | 3.1.1 统一检测模型设计 | \(\mathcal{L}_{\text{CIoU}} = 1 - \rho + \frac{d^2}{c^2} + \alpha v\) | CIoU 边界框回归损失 |
| (3-2) | 3.1.2 单目测距与安全间距判定 | \(D = \frac{f \cdot H_{\text{real}}}{h_{\text{pixel}}}\) | 单目测距（相似三角形） |
| (3-3) | 3.1.2 单目测距与安全间距判定 | \(X = \frac{(u - c_x) \cdot Z}{f_x},\; Y = \frac{(v - c_y) \cdot Z}{f_y}\) | 像素坐标到相机坐标系 |
| (3-4) | 3.1.2 单目测距与安全间距判定 | \(d_{12} = \sqrt{(X_1 - X_2)^2 + (Y_1 - Y_2)^2 + (Z_1 - Z_2)^2}\) | 两气瓶欧氏距离 |
| (3-5) | 3.2.1 火焰与多帧一致性判定 | \(\text{告警触发} \Leftrightarrow \exists t,\; \forall k \in \{0,\ldots,N-1\},\; p_{\text{fire}}(t-k) \geq \tau\) | 多帧一致性触发条件 |
| (3-6) | 3.2.2 安全帽佩戴判定 | \(\text{已佩戴} \Leftrightarrow x_1^p \leq x_c \leq x_2^p \,\wedge\, y_1^p \leq y_c \leq y_h\) | 安全帽中心点落入头部区域 |
| (3-7) | 3.2.3 人脸授权与闯入检测 | \(s_{ij} = \frac{\mathbf{f} \cdot \mathbf{f}_j}{\|\mathbf{f}\| \|\mathbf{f}_j\|}\) | 人脸特征余弦相似度 |
| (3-8) | 3.3.1 低照度图像增强 | \(s_k = T(r_k) = (L-1) \sum_{i=0}^{k} p_r(r_i) = (L-1) \cdot \text{CDF}(r_k)\) | 直方图均衡（CDF 映射） |

共 **8** 个公式，编号 (3-1)～(3-8)。
