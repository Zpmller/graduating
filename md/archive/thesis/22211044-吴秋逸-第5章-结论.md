# 5 结论

本文面向矿业动火作业安全管控需求，以国家标准 GB 30871—2022[1] 等规范为约束，设计并实现了一套基于 AI 视觉的矿业动火作业行为安全监测系统，实现了对气瓶安全距离、火焰/火星、人员安全帽佩戴及作业人员身份授权的多目标协同感知与实时预警，形成了“视频采集—智能分析—决策—告警”的闭环监测体系。

**主要工作与结论概括如下：**

（1）**需求与系统设计**：在分析动火作业风险与规范要求的基础上，明确了气瓶间距、火焰/火星、安全帽佩戴、人脸授权、低照度适应及告警联动等核心功能需求；采用边缘—云端分层架构，边缘端负责视频拉流与 AI 分析，云端负责设备管理、告警存储与 Web 管理端展示，数据流与安全事件抽象统一，子系统间通过 REST API 与设备令牌/JWT 认证对接，便于扩展与维护。

（2）**多目标检测与气瓶安全距离**：采用 YOLOv8/YOLOv11 单阶段检测模型对氧气瓶、乙炔瓶、火焰/火星、安全帽及人员等进行统一检测（8 类），在单目视觉下结合针孔模型与相机标定实现气瓶空间位置估计与欧氏距离计算（式 (3-2)～(3-4)），通过时间滤波提升距离判定稳定性，并依据规范对氧乙炔瓶间距小于 5 m 的情形触发告警。实验表明，检测与测距性能满足现场实时监测与安全间距判定的需求。

（3）**火焰、安全帽与人脸授权**：通过多帧一致性判定（式 (3-5)，\(N=3\)）降低火焰/火星误报；通过人员框与安全帽框的空间关联（式 (3-6)）判定佩戴状态；采用深度人脸特征（ArcFace/DeepFace）与授权库的余弦相似度比对（式 (3-7)）实现身份核验与非授权闯入检测。在低照度场景下引入 CLAHE 对 LAB 的 L 通道进行直方图均衡（式 (3-8)），保障检测与识别的稳定性。各模块输出统一抽象为安全事件，经规则引擎触发声光告警与管理端展示，模块化与事件驱动设计使系统具备良好的可扩展性。

（4）**系统验证**：在给定实验环境与多源数据集下，对多目标检测精度、气瓶距离估计与安全间距判定、火焰多帧判定与安全帽及人脸模块进行了测试；端到端功能与告警联动、长时间运行稳定性及可扩展性均达到预期，表明所建系统能够为矿山动火作业提供可落地的智能监测技术方案，提升安全管控的客观性与实时性。

**不足与展望：**

（1）单目测距在标定误差或安装晃动较大时距离估计会受影响，后续可结合深度传感器或更稳健的标定与滤波策略进一步优化。（2）人脸授权在极端低照度、大角度或严重遮挡下识别率仍有提升空间，可引入专用低照度人脸增强与多帧融合策略。（3）当前实验多在模拟或有限现场条件下开展，后续可在更多矿山现场部署试用，收集长期运行数据与反馈，优化模型与规则参数。（4）可进一步将告警与作业审批、人员考勤等业务流程对接，形成更完整的智慧安全管控平台。

综上所述，本文完成的矿业动火作业行为安全监测系统实现了多风险协同感知与规范量化判定，具备较好的工程可部署性与可扩展性，对矿山动火作业的智能化安全监测具有实际应用价值与参考意义。
